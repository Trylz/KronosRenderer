Features

Nebula is a fast and unbiased physically based renderer for Windows.
Its simple user interface let you to create amazing shots with a minimal iteration time.

Realtime raytracing

The realtime progressive raytracing mode allow you to interactively edit scenes and see a preview of the final result. Only start a final render when you have found the perfect spot.

  • Rendering in the main viewport
  • Live materials and lights edition
  • Fast feedback

Intuitive interface

The user interface comes with a collection of helpers to simplify scene manipulations.
Modify the different elements in just one-two clicks.

  • Simple to use
  • Diversified tools
  • High fidelity realtime viewport

Photorealistic renders

From subsurface scattering to participing media, Nebula supports a high range of lightning effects to help you to create stunning images.

  • Fast
  • Unbiased
  • CPU and hybrid GPU modes
  • Post-process image editor
